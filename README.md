
## Requisitos

- Python 3.x
- Biblioteca `ipaddress`


## Como usar


1. Clona este repositorio
```bash
  git clone https://github.com/shrek.mp4/net-multitool.git

```
2. Instala las Bibliotecas
```bash
  pip install ipaddress

```
3. Ejecuta el programa
```bash
  python net-multitool.py

```
